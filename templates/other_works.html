Pareto Fronts of NSGA - II with Java

Non Dominated Sorting Genetic algorithm is a multi-objective genetic algorithm. This work specifically implementing NSGA - II with Java for 2 objectives. Since the implementation and tracking Pareto fronts for each generation was the focus, objective functions used were x2 and (x-2)2 for a population of 10,000 for 10 generations. The crossover probability and mutatin probability used are 0.98 and 0.03 respectively. The algorithm uses a non-dominating sorting algorithm based on the domination ranks. Fitness function is calculated by applying Min-Max Normalization on the genetic code.

Analysis on Laser Designated Pod data in aircrafts

Even the single flight testing of an aircraft provides data in large quantity which need to be thoroughly analysed, every single time. The Laser Designated Pod in an aircraft works to track and identify objects by firing up a laser. The pod aims to track and identify the objects and their location in the air space. At Mission and Combat System Research and Design Centre, the application, after simulating the work of Laser Designated Pod and retrieveing raw data, converted the basic raw unreadable data to decimal and readable format. For enhanced analysis, an additional tool within the application offered various choices for the relevant various fields to be portrayed graphically, against time or each other. The summarized details are:

  • Application was based on GUIDE (Graphical User Interface DEvelopmnet) – a MATLAB GUI application.
  • The data conversion to readable set was institution’s defined format.
  • The application improved the entire analysis by 5% and achieved nearly 80% accuracy; also reduced 6 daily manual hours.

Leaf Constrained MST on Artificial Bee Colony

Implemented Artificial Bee Colony Evolutionary Algorithm to develop a LC-MST – discrete optimization problem when fed random co-ordinates of vertices and the defined constraint as the number of leaves, in the polynomial time complexity. The project aimed to contrast the results by implementing heuristics and ABC to identify the LC-MST for a predefined leaf constraint for the MST. ABC certainly helped in improving the identifying LC-MST in terms of time up to a relative number of leaves(n) corresponding to the size of the graph. However, as n increased, heuristics showed better performance.
