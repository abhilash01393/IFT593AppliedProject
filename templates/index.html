Deep Learning with Sight and Vision

Created by Abhilash Chaudhary


An innovative way to read and write sign language to a digital computer system.

Simple reduction of complex mode of sign communication by classifying images into 29 classes.

By Supervised Machine Learning with multi layered Convoluted Neural Network (CNN).

Model with an accuracy of

0%

at an average while detecting the sign in the image which is based on trained data.

Keras + Python


Model structured in the ML libraries of Python 3.6.7 and the neural network is build in the extensible Keras 2.2.4.

TensorFlow and SciKit Learn


Running Keras on TensorFlow 1.12.0 and SciKit Learn 0.20.1 developed the targeted accurate model.

Data Set

The data set is a collection of images of alphabets from the American Sign Language, separated in 29 folders which represent the various classes. The training dataset contains 87,000 images which are 200x200 pixels. There are 29 classes, of which 26 are for the letters A-Z and 3 classes for SPACE, DELETE and NOTHING. These 3 classes are very helpful in real-time applications and classification. The test dataset contains a mere 29 images, to encourage the use of real-world test images.
